计算机体系结构/内存分层体系

计算机体系结构

计算机的体系结构主要包含三大块内容：
CPU 内存 IO

内存的分层体系

内存的层次结构：

cpu可以访问的指令/数据所处的位置可以是上图所示的四个部分

  • 寄存器
  • cache
  • 主存
  • 磁盘（虚拟内存）

寄存器和cache是cpu内部的,操作系统无法直接进行管理,速度很快,但空间很小

主存也叫物理内存,主要存放操作系统本身和运行的代码,比寄存器和cache大很多,但没有那么快
一掉电 数据就没了

磁盘大很多,但是速度要慢很多

整体形成类似金字塔的层次结构

操作系统完成的事情

四个重点完成的目标

  • 抽象 逻辑地址空间
  • 保护 独立地址空间
  • 共享 访问相同内存
  • 虚拟化 更多的地址空间
    (主存,硬盘是物理地址空间,而程序运行时看到的则是逻辑地址空间)

在操作系统中管理内存的不同方法

  • 程序重定位
  • 分段
  • 分页
  • 虚存管理
  • 按需分页虚拟内存

实现高度依赖于硬件

  • 必须知道内存架构
  • MMU(内存管理单元): 硬件组件负责处理CPU的内存访问请求CPU
