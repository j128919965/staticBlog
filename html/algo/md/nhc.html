拿火柴问题

现有21根火柴，两人轮流取，每次可以取走1-4根，不可多拿或少拿。
请编写一个程序进行人机对弈，要求人先，计算机后，拿走最后一根的人输。计算机为常胜将军。

这道题考虑最少的情况。如果当前剩下1根，人先取，他只能拿一根，那这样他就输了。
也就是说，只要计算机给人留下一根，那人就必败了。
因为总数是21根，所以只要每一轮先手和后手拿到的总量是5，那么最后给先手剩下的就一定是1。
（人 类 一 败 涂 地）
（但是如果总量不是5n+1，先手的一方就可以把当前总数变成5n+1，这样后手就必败了。）
代码：

import java.util.*;

public class Solution {

    public static void main(String[] args) {
        int n = 21;
        Scanner sc = new Scanner(System.in);
        int p1takes=0;
        int p2takes=0;
        while(n>0){
            System.out.println("\n当前剩余火柴数量："+n);
            while (p1takes>n||p1takes<1||p1takes>4){
                System.out.print("请玩家拿火柴，请输入您要拿的的数量(1-4)：");
                p1takes = sc.nextInt();
            }
            System.out.println("您拿了"+p1takes+"根火柴！");
            n-=p1takes;
            if(n==0){
                System.out.println("您失败了！");
                return;
            }
            System.out.println("请计算机拿火柴");
            p2takes = 5-p1takes;
            System.out.println("计算机拿了"+p2takes+"根火柴！");
            n-=p2takes;
            if(n==0){
                System.out.println("您胜利了！");
                return;
            }
            p1takes=0;
        }
    }
}
